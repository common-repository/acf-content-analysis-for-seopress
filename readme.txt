WordPress plugin that adds the content of usable content ACF fields to the SeoPress score analysis.

== Description ==

This plugin ensures that [SeoPress](https://wordpress.org/plugins/wp-seopress/) analyzes all [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/) content.  This plugin hook into the analyser of SeoPress, explained on this [SeoPress-page](https://www.seopress.org/support/guides/add-acf-custom-fields-to-content-analysis/).

== Frequently Asked Questions ==

= Required plugins? =

This plugin requires you are using SEOPress or SEOPress Pro and ACF or ACF Pro

= Which fields will be indexed? =

These ACF field types will be indexed: WYSIWYG, text, text area, flexible content, group and repeater.
